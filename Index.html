Déplacement ITRF avec le client MapQuest

La carte ci-dessous est le fruit d'un projet de web mapping, fait à l'Ecole Nationale des Sciences Géographiques (ENSG). Il représente des données fictives semblables à des données de déplacements. Le but de ce projet étant de représenter des données de déplacements verticaux et horizontaux des stations ITRF.

Légende

Les stations ITRF sont sous formes de cercles, le code couleur représente les déplacements verticaux, rouge étant les déplacements les plus descendants pour aller jusqu'au bleu les déplacements les plus ascendants. Les déplacements horizontaux sous formes de segments issus du centre de leur cercle respectif.
